# 21장 빌트인 객체

## 21.1 자바스크립트 객체의 종류

| 종류 | 설명 |
|---|---|
| 표준 빌트인 객체 | ECMAScript 사양에 정의된 객체 (Object, String, Number 등) |
| 호스트 객체 | 실행 환경에서 제공하는 객체 (브라우저: DOM, fetch 등 / Node.js: fs 등) |
| 사용자 정의 객체 | 사용자가 직접 정의한 객체 |

## 21.2 표준 빌트인 객체
Object, String, Number, Boolean, Symbol, Date 등 40여 개의 표준 빌트인 객체가 있다
Math, Reflect, JSON을 제외한 표준 빌트인 객체는 모두 인스턴스를 생성할 수 있는 생성자 함수다

## 21.3 원시값과 래퍼 객체
래퍼 객체: 원시값(문자열, 숫자, 불리언)에 대해 객체처럼 접근하면 생성되는 임시 객체

```javascript
const str = 'hi'
console.log(str.length) 
// 1. 'hi'가 임시로 String 래퍼 객체로 변환
// 2. String.prototype.length 사용
// 3. 처리 끝나면 다시 원시값으로 되돌림
// 4. 래퍼 객체는 가비지 컬렉션 대상
console.log(typeof str)  // string
```

```javascript
const str = 'hi'
str.name = 'Lee'          // 래퍼 객체에 name 추가
// → 처리 끝나면 이 래퍼 객체는 버려짐(가비지 컬렉션)
console.log(str.name)     // undefined ← 새 래퍼 객체에는 name이 없음
```

## 21.4 전역 객체
전역 객체는 표준 빌트인 객체, 호스트 객체, var로 선언한 전역 변수와 전역 함수를 프로퍼티로 갖는다
(프로토타입 상속 관계상 최상위라는 뜻은 아님)

```javascript
var foo = 1
console.log(window.foo) // 1 (var 전역 변수 → 전역 객체 프로퍼티)
```

### 21.4.1 빌트인 전역 프로퍼티

| 프로퍼티 | 설명 |
|---|---|
| Infinity | 무한대를 나타내는 숫자값 |
| NaN | 숫자가 아님을 나타내는 숫자값 |
| undefined | 원시 타입 undefined |

### 21.4.2 빌트인 전역 함수

| 함수 | 설명 |
|---|---|
| eval | 문자열을 코드로 실행 (**사용 금지** - 최적화 안 됨, 보안 위험) |
| isFinite | 유한수면 true, 무한수/NaN이면 false |
| isNaN | NaN이면 true |
| parseFloat | 문자열 → 부동 소수점 숫자로 변환 |
| parseInt | 문자열 → 정수로 변환 |
| encodeURI / decodeURI | 완전한 URI 인코딩/디코딩 |
| encodeURIComponent / decodeURIComponent | URI 구성 요소 인코딩/디코딩 |

### 21.4.3 암묵적 전역
선언하지 않은 변수에 값을 할당하면 암묵적으로 전역 변수처럼 동작
```javascript
function foo(){
    y = 20  // 선언 없이 할당 → 암묵적 전역
}
foo()
console.log(y) // 20
```
- y는 변수가 아니라 전역 객체의 프로퍼티 → 호이스팅 안 됨
- delete 연산자로 삭제 가능 (진짜 변수는 delete 불가)
